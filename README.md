# 🟡 Backrooms Escape

> **"Tente escapar desse monstro peludo."**

Você não sabe como chegou aqui.
Corredores infinitos, paredes amareladas, um silêncio estranho…
e **algo está te observando**.


---

## 🎮 Sobre o jogo

**Backrooms Escape** é um jogo simples de terror em primeira pessoa, rodando direto no navegador.

Seu objetivo é um só:

👉 **encontrar a saída antes que o monstro te encontre.**

Mas cuidado…
ele também está procurando você.

---

## 🧠 Mecânicas

* 🕹️ Movimento em primeira pessoa (WASD + mouse)
* 🧱 Labirinto fechado
* 👁️ Atmosfera tensa estilo backrooms
* 👹 Monstro perseguidor (não pare pra olhar muito…)

---

## ⚠️ Regras simples

* Se ele te alcançar → **GAME OVER**
* Não existe mapa
* Não existe ajuda
* Só existe você… e ele

---

## 🚀 Como jogar

1. Acesse o jogo:

   ```
   https://gabrielcardoso25.github.io/backrooms/
   ```

2. Clique na tela para iniciar

3. Use:

   * **W A S D** → mover
   * **Mouse** → olhar ao redor

4. Boa sorte.
